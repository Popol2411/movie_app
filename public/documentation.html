Popol´s Movie Database

This movie database offers the possibility to search for movies and inform the user about the actors, genre, director.

ENDPOINT RETURN ENDPOINT URL HTTP METHOD TYPE QUERY PARAMETER REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Return a list of ALL movies to the user /movies GET None None JSON object with all movies data
Return data about a single movie by title to the user /movies/[title] GET :title None JSON object of the selected movie with following information: 
        
        {
        "Title": "Saving Private Ryan",
        "Genre": {
                  "Name": "War"
                  },
        "Director": {
                      "Name": "Steven Spielberg",
                      "Born": "1946"
                    }
        },
Return data about a genre by name/title (e.g. "Thriller") /movies/genre/[genre] GET :genreName None JSON object of the selected genre with following information: 
        
        {
        "Name": "Comedy"
        }
Return data about a director (e.g. Birth Year) by name /movies/director/[director] GET :directorName None JSON object of the selected director with following information: 
        
        {
        "Name": "Steven Spielberg",
        "Born": "1946"
        }
Allow new users to register /users POST None JSON object with the data to be added to the user: 
        
        {
        "Name": "Jean-Paul Belmondo",
        "favoriteMovies": ["Forrest Gump", "Titanic", "Jurassic Park"]
        }
JSON object with the data added and new user created (id generates automatically): 
        
        {
        "Name": "Jean-Paul Belmondo",
        "favoriteMovies": ["Forrest Gump", "Titanic", "Jurassic Park"],
        "id": 12345
        }
Allow users to update their user info (username) /users/[id] PUT :id JSON object with the new username: 
        
        {
        "Name": "Jean-Paul Gaultier",
        }
JSON object with the username updated: 
        
        {
        "Name": "Jean-Paul Gaultier",
        "favoriteMovies": ["Forrest Gump", "Titanic", "Jurassic Park"],
        "id": 12345
        }
Allow users to add a movie to their list of favorites /users/[id]/[movieTitle] POST :id, :movieTitle None `${movieTitle} has been added to user ${id}´s array`
Allow users to delete a movie from their list of favorites /users/[id]/[movieTitle] DELETE :id, :movieTitle None `${movieTitle} has been removed from user ${id}´s array`
Allow users to delete their profile /users/[id] DELETE :id None `User ${id} has been deleted`
